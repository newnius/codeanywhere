# Go lang

## Download
[download](https://golang.org/dl/)

## Extract
```bash
sudo tar -C /usr/local -xzf go$VERSION.$OS-$ARCH.tar.gz
```

## Export Path
```bash
sudo bash -c 'cat >>/etc/profile <<EOF
export PATH=\$PATH:/usr/local/go/bin
EOF'
```
